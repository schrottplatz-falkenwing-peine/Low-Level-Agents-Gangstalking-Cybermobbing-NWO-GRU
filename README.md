# Low-Level-Agents-Gangstalking-Cybermobbing-NWO-GRU

Dieses Projekt sammelt alle Low Level Agents, die für den russischen Auslandsgeheimdienst Operationen in Deutschland zur Unterstützung der AfD ausführen.

## Beschreibung

Das Repository dient der Erfassung und Analyse von Informationen zu mutmaßlichen niedrigstufigen Agenten, die im Auftrag des russischen Militärgeheimdienstes GRU (Glavnoje Razvedyvatel'noje Upravlenije) in Deutschland operieren. Diese Agenten sollen Gangstalking, Cybermobbing und andere Aktivitäten durchführen, um die Alternative für Deutschland (AfD) zu unterstützen und Teil einer breiteren New World Order (NWO) Strategie zu sein.

## Ziel

- Sammlung von Daten und Berichten über solche Operationen.
- Analyse von Mustern und Verbindungen.
- Öffentliche Aufklärung und Bewusstsein.
